# kalokairi

・Qt, Dx11, C++, HLSL  
・モデリングツール  
・アニメーションツール  
・ディファードレンダリング  
・トゥーンレンダリング  
・Z値、法線エッジ  
・ジオメトリ生成（mesh field, box, sphere, geosphere, capsule)  
・階層アニメーション  
・複数アニメーション同時再生、ブレンド  
・足を出した時のみ前進  
・アニメーション中にパーツへのTransform割り込み  
・ディザ抜き  
・シャドウマップ
